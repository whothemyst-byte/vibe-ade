FAQ

Does Vibe-ADE work offline?

Yes for local routing via Ollama. Cloud routing requires internet and a valid API key.

Which platforms are supported?

Current release target is Windows desktop.

Where do I report beta issues?

Use the GitHub repository issues tab to report crashes or UX bugs.

How do updates work?

Check release notes and download the latest installer from the release tag.

Ready to test Vibe-ADE?
